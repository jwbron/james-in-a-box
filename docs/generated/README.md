# Generated Documentation

This directory contains machine-readable documentation indexes generated by automated analysis tools.

## Files

| File | Description | Generator |
|------|-------------|----------|
| `codebase.json` | Structured codebase analysis (components, files, relationships) | Codebase Analyzer |
| `patterns.json` | Extracted code patterns and conventions | Pattern Extractor |
| `dependencies.json` | Internal and external dependency graph | Dependency Analyzer |

## Purpose

These files enable LLMs to efficiently query specific information without loading entire documentation:

```bash
# Example: Get info about a specific component
jq '.components[] | select(.name == "GitHubWatcher")' codebase.json

# Example: Find files using a pattern
jq '.patterns.async_processing.examples' patterns.json

# Example: Get dependencies for a component
jq '.dependencies.internal.SlackNotifier' dependencies.json
```

## Generation Schedule

| Index | Frequency | Trigger |
|-------|-----------|--------|
| `codebase.json` | Weekly | Also on significant code changes |
| `patterns.json` | On detection | When new patterns identified |
| `dependencies.json` | Weekly | Also on dependency changes |
