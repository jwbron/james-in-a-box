# Coding Standards

**Tech**: Python/Django, React/TypeScript, Jest/PyTest, PostgreSQL/Redis

**Python**: PEP 8, type hints, docstrings, explicit imports
**JS/TS**: TypeScript for new code, ESLint, functional components, `const` over `let`

**Commands**: `make serve`, `make test`, `make lint`, `make fix`

**Security**: No secrets in commits, validate input, sanitize output, HTTPS only
